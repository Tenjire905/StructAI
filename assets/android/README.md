# Android Tab Icons (NativeTabs Fallback)

Minimal PNG fallbacks for `NativeTabs.Trigger.Icon` `src` prop when Material Symbols (`md`) are unavailable (e.g. older dev clients).

| Datei | Tab |
|-------|-----|
| `tab-akademie.png` | Akademie |
| `tab-lab.png` | Prompt Lab |
| `tab-profil.png` | Profil |

**Primary path (SDK 56):** `md` Material Symbols in `_layout.tsx` — no asset required for normal dev builds.

Specs: 24×24 PNG, single-color template-friendly glyph on transparent background.
